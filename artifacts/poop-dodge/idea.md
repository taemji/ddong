# 똥피하기 — 레트로 도트 아트 생존 게임

## HMW

> "얼마나 버틸 수 있는지"의 긴장감이 한 판 안에서 계속 고조되는 경험을 만들 수 있을까?

## 핵심 루프

1. 3레인 중 하나에 캐릭터 위치
2. 위에서 똥이 랜덤 레인으로 떨어짐
3. 방향키(PC) / 탭(모바일)으로 레인 이동해 피하기
4. 시간이 지날수록 속도 증가, 다중 똥 동시 낙하
5. 맞으면 게임오버 → 생존 시간 + 점수 표시

## MVP 범위

- 게임 캔버스 (HTML Canvas + React useRef)
- 캐릭터 이동 — 키보드 + 모바일 터치
- 똥 생성 + 낙하
- 충돌 판정 (순수 함수로 분리)
- 점수 / 게임오버 화면
- 도트 아트: Canvas drawRect 픽셀 단위 렌더링

## Not Doing (v1)

- 온라인 랭킹
- 사운드 효과
- 아이템 / 파워업
- 캐릭터 선택

## 기술 결정

| 항목 | 결정 |
|---|---|
| 페이지 | `app/game/page.tsx` |
| 렌더링 | HTML Canvas + `requestAnimationFrame` |
| 상태 관리 | `useRef` (게임 루프), `useState` (UI) |
| 입력 | `keydown` + `touchstart` |
| 테스트 | 충돌 판정 순수 함수 → Vitest |
| 플랫폼 | 브라우저 PC + 모바일 |

## 숨은 가정

| 가정 | 분류 |
|---|---|
| 3레인 이동만으로 충분히 재미있다 | **Must Be True** |
| 속도 증가로 긴장감이 고조된다 | **Should Be True** |
| 고득점 공유 기능이 재플레이를 유도한다 | Might Be True |
